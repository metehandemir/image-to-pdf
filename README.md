# Images to PDF

A simple, open-source tool to convert images into a single PDF file. Everything runs locally in your browser — no files are uploaded to any server.

## Features

- **Drag & drop** or browse to add images
- **Reorder** images by dragging or using up/down arrows
- **Preserves original dimensions** — each PDF page matches the image size
- **Supports** PNG, JPG, JPEG, WEBP
- **Dark mode** support
- **No server uploads** — fully client-side

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- [Next.js](https://nextjs.org)
- [Tailwind CSS](https://tailwindcss.com)
- [jsPDF](https://github.com/parallax/jsPDF)
